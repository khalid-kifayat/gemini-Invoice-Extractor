## Installation 🛠️

1. **Clone the Repository** 📥:
 ```bash
 git clone https://github.com/username/gemini-Invoice-Extractor.git
 cd gemini-Invoice-Extractor
 ```

2. **Install Dependencies** 📦:
 ```bash
 pip install -r requirements.txt
 ```

## Usage 🎯

1. **Run the Application** 🚀:
 ```bash
 python app.py
 ```
